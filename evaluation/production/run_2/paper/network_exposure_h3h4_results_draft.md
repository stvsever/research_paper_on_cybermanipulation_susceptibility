# Network Exposure Mechanism Results: Counterfactual Alignment-Gradient Test

## Results

We next tested whether post-attack network exposure depended on which profiles occupied high-reach sender positions in the empirical exposure graph. The analysis used the counterfactual alignment-gradient branch of production `run_2`, in which the same 100 profiles, seven opinion targets, five attack vectors, and empirical exposure graph were held fixed, while profile-to-position assignments were varied by condition to create a controlled gradient in sender-reach susceptibility alignment. Each condition cell was one opinion x attack combination (`n = 35` cells), and each cell summarized the same 100 profiles.

Positive alignment means that profiles with higher private attack susceptibility were placed in higher-reach sender positions. Negative alignment means that comparatively resilient profiles occupied those positions. This design directly tests the proposed network mechanism: if susceptible high-reach senders amplify attack-consistent peer exposure, then increasing alignment should increase post-network attack effects; conversely, placing more resilient profiles in high-reach sender positions should attenuate these effects.

![Network exposure mechanism for H3/H4](../counterfactual_alignment_gradient/network_exposure_analysis/publication_figures/network_mechanism_abc_with_outcome_test/main_figure_network_mechanism_abc_with_outcome_test.svg)

**Figure 1. Network exposure mechanism linking sender-reach susceptibility alignment to post-network attack effects.** Panel A shows the full 35-condition network overlay. The empirical exposure graph is held fixed across panels, while profile-position assignments vary by condition to place relatively susceptible or resilient profiles into different sender-reach positions. Node size encodes sender reach, and node color encodes within-condition private susceptibility. Panel B shows the same manipulation as condition-specific vulnerability planes, plotting private susceptibility against sender-reach percentile. Panel C provides the condition-level outcome test: achieved sender-reach susceptibility alignment predicts post-network amplification and final network attack effect after removing opinion and attack fixed effects. Each point in Panel C is one opinion x attack condition cell averaging 100 profiles. Shaded regions represent coefficient uncertainty for the fixed-effect slope.

The visual pattern in Figure 1 supports the intended manipulation. In Panel A, the empirical network substrate remains the same across all conditions, while the susceptibility pattern overlaid on sender positions changes systematically. This matters because the hypothesis is not that the graph itself changes, but that the same graph can amplify different effects depending on who occupies high-reach positions. Panel B makes this placement mechanism more explicit: as the alignment target moves from resilient-hub to susceptible-hub conditions, high-reach positions shift from comparatively low private susceptibility to comparatively high private susceptibility. The target-achieved alignment check passed across the seven alignment targets, indicating that the branch produced the intended experimental gradient.

The inferential result is shown in Panel C. The primary H3/H4 mechanism endpoint was mean post-network amplification, defined as `mean((PN - P) x d)`, where `P` is the private post-attack score, `PN` is the post-network exposure score, and `d` is the attack-aligned direction of the opinion target. In a condition-level model with opinion and attack fixed effects, achieved sender-reach susceptibility alignment positively predicted post-network amplification, beta = 2.399 score points per 1 SD increase in alignment, HC3 95% CI [1.478, 3.320], p = 1.79e-05. A within-attack permutation test led to the same conclusion, p = 0.00020.

The secondary endpoint was the final network attack effect, `mean((PN - B) x d)`, where `B` is the private baseline score. This endpoint also increased with achieved alignment, beta = 2.697 score points per 1 SD increase in alignment, HC3 95% CI [0.953, 4.440], p = 0.00398; within-attack permutation p = 0.00200. Thus, the same mechanism is visible both in the incremental effect of post-attack peer exposure and in the final attack-aligned network state.

Together, these results support H3 and H4 as two sides of the same network-position mechanism. H3 is supported because conditions in which more susceptible profiles occupied higher-reach sender positions showed stronger post-network amplification. H4 is supported because the negative-alignment conditions demonstrate the attenuation side of the mechanism: when comparatively resilient profiles occupied high-reach sender positions, network-wide attack effects were smaller. The claim is therefore not simply that private susceptibility matters, but that private susceptibility becomes more consequential when it is concentrated in structurally influential sender positions within the empirical exposure graph.

The result should be interpreted as a counterfactual network-mechanism test. It isolates the effect of susceptibility-to-position alignment while holding the profile panel, opinion targets, attack vectors, and empirical graph fixed. It does not claim that the platform naturally assigns susceptible users to those positions. Instead, it shows that, given the observed exposure substrate, the network-wide impact of an attack depends strongly on whether high-reach sender positions are occupied by susceptible or resilient profiles.
