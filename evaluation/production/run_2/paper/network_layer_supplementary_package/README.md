# Network-Layer Supplementary Package

This folder is a self-contained staging package for supplementary network-layer analyses from production `run_2`.

It contains:

- `figures_png/`: PNG-only candidate supplementary figures.
- `tables/`: CSV tables that support the network-layer methods, H5 profile-level bridge, and H6/H7 network-position mechanism.

The package is intentionally curated. It excludes raw LLM outputs, raw stage folders, failed-run artifacts, individual condition-panel PNGs, and redundant diagnostics that would weaken the supplement by volume rather than strengthen the paper.

## Recommended Use

Use the **Core** items as the main supplementary network-layer package. Use **Optional** items only if the supplement needs additional diagnostic depth or source-data transparency.

The main paper already contains the final profile direct-vs-network figure and the final H6/H7 composite mechanism figure. This package therefore focuses on supplementary evidence, expanded component figures, exact tables, source-data tables, and robustness diagnostics.

## Test-Retest Stability Subsection

Use this subsection first if the supplement needs to establish that the network-layer measurements and mechanism are stable under focused retest. The retest uses the first 30 retested profiles and compares original Run 2 values with focused retest values for baseline network exposure (`BN`), post-attack network exposure (`PN`), derived deltas, ranking stability, and the sender-reach alignment signal.

## Items Deliberately Excluded

- Raw LLM prompt/response files and raw stage output folders.
- `condition_stage04b_outputs_failed_sandbox_dns_20260622T0207Z`, because it is an execution-process artifact rather than a scientific result.
- Individual per-condition vulnerability-plane PNGs, because the full-grid figure is clearer and complete.
- SVG, PDF, and TIFF variants, because this package is explicitly PNG-only for figures.
- The final main-paper figures, because they already live in the official paper assets; this folder is for supplementary analyses and source support.
